# android-pc-controller
control a pc via an android phone with an arduino

# ELECTRONICS
ardino micro\
HC-06

# BLUETOOTH TRANSMITION PARAMATERS
`type <string>;`\
`move <x-movement> <y-movement> <scroll_value> <mouse_button(1 == left click)(2,3 == right click)(4 middle click)`\

# WIRING
RXD --> 10\
TXD --> 11\
GND --> GND\
VCC --> 5V
